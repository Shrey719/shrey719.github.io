# So whats a bootloader

Because backwards compatibility, it is a right pain in the ass to get modern features working - so thats the first thing I did

A bootloader sounds fancy, but in reality its any file that is 512 bytes long, runs in an infinite loop, and has the last two bytes as the magic number 0xaa55
A simple bootloader could be something like

org 0x7c00

bits 16
jmp $
times 510-($-$$) db 0
dw 0xaa55

Since it would be complicated to explain how the GDT and Long mode work, im not going to (am lazy)

Basicaly, those two allow you to use stuff like 32 and 64 bits and a higher level langauge like C
